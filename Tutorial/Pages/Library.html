Library
The SmallJS standard library is located in the SmallJS repo folder: ./Smalltalk .
Every class is implemented in a separate file with extension: *.st .
Further subfolders are used to group classes logically,
but their location does not affect their use.
All classes are unique global variables.

You can browse the documentation of all classes in the class Reference.

Method names in SmallJS are kept mostly the same as their JS equivalents.
So you can also browse the Mozilla MDN documentation for implementation details.

The remainder of this tutorial section focuses on using basic library functionality,
Things that are often hardcoded in other languages, like strings, numbers and collections.
So only limited subset of all classes.

Examples here can be evaluated in the Playground on this site.
