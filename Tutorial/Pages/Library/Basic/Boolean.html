Boolean
A Boolean refers to either the true or false singleton (only) object.
Keywords true and false can be used in soure code to refer to these objects.
Testing if an object is true or false can be done with methods
ifNil: [ ... ], ifFalse: [ ... ] or a combination of them.
The method not flips the true/false value of a Boolean.

Note:
ST does not coerce 'truthy' values into booleans, like integers 1 and 0 .
Conversions to booleans must be done explicitly.

Examples:
  • true ifTrue: [ 'yes' ] returns 'yes' .
  • false ifTrue: [ 'yes' ] ifFalse: [ 'no' ] returns 'no' .
  • true not = false returns true .
  • 1 ifTrue: [ 'no-no' ] returns 1 (block not evaluated).
