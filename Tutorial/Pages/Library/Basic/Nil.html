Nil
The class Nil has a singelton (only) instance object called nil.
Keyword nil can be used in ST soure code to refer to this object.
Variables are initialised to nil by default.
Testing if an object is nil or not nil
can be done with methods ifNil: [ ... ] or ifNotNil: [ ... ] .

Examples:
  • | a | a = nil returns true .
  • nil ifNil: [ 7 ] returns 7 .
  • nil ifNotNil: [ 8 ] returns nil .
Advanced
The nil object also stores JS null internally,
so it can be passed to and returned from INLINE JS code
that takes null as arguments or returns it.
