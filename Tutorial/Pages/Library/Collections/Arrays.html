Arrays
The class Array implements the most commonly used collection type.
Arrays contain collections of element objects.
Arrays can be defined in code with : #( ... ),
specifying space separated objects or expressions within the parentheses.
An Array can be allocated with a specific size with the class method new: , with elements set to nil by default.
Arrays can grow dynamically by adding elements.

Examples:
  • #( 3 'd' ( 3 + 0.5 ) ) returns #( 3 d 3.5 ) .
  • Array new: 2 returns #( nil nil ) .
  • Array new add: 4; add: 5 returns #( 4 5 ) .
Accessing elements
Array elements can be:
- Retrieved with a zero-based index using at:.
- Replaced using at:put.
- Iterated over using do:.
- Converted into a new array using map:.

Examples:
  • #( 3 4 5 ) at: 1 returns 4 .
  • #( 3 4 5 ) at: 1 put: 6 returns #( 3 6 5 ) .
  • #( 3 4 5 ) do: [ :e | Console log: e ] logs numbers 3, 4 and 5 on the console.
  • #( 3 4 5 ) map: [ :e | e squared ] returns #( 9 15 25 ) .
Check out the Array class reference for more methods.
Typed arrays
Typed arrays are subclasses of the class: TypedArray .
These arrays store one specific data type compactly.

For handling binary data there are the classes Uint8Array and ArrayBuffer .
For use with AI models, there is the class: Float16Array .
