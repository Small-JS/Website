Dictionaries
The class Dictionary implements a general purpose key-value dictionary.
A key can be any object that can be compared for equality,
but mostly magnitudes are used that are compared by value, like strings and integers.
Looking up a non existent key throws an error, unless there is an absent clause.

Examples:
  • | dict | dict := Dictionary new at: 5 put: 'e'. returns Dictionary( { 5, e } )
    Adding to the previous line, using the variable dict , you can evaluate:
  • dict at: 5 returns e .
  • dict at: 9 throws error Key not found: 9 .
  • dict at: 9 ifAbsent: [ nil ] returns nil .
  • dict at: 5 put: 'f' returns Dictionary( { 5, f } ) (value modified for key).
Map
The class Map implements a dictionary where keys must be strings.
It encapsulates the JS class Map that performs better than class Dictionary .
JS Map method names are preserved, that differ from methods in the class Dictionary above.
Looking up a non-existent key returns nil, not an error.

Examples:
  • | map | map := Map new set: 'a' value: 1. returns Map( { a, 1 } )
    Adding to the previous line, using the variable map , you can evaluate:
  • map get: 'a' returns 1 .
  • map get: 'z' returns nil .
  • map set: 'a' value: 2 returns Map( { a, 2 } ) (value modified for key).
