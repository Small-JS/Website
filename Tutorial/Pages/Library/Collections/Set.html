Set
The class Set implements mathematical sets.
A set contains a collection of element values that can occur only once.
Looking up a non-existent element value returns nil.

Examples:
  • | set | set := Set new add: 3; add: 4. returns Set( 3 4 ) .
    Now with variable set :
  • set add: 4 returns Set( 3 4 ) (nothing was added).
  • set size returns 2 .
  • set has: 4 returns true .
  • set find: 5 returns nil .
  • set forEach: [ :value | Console log: value ] logs 3 and 4 on the console.
Sets support mathematical set operations like intersection, difference and union.

Examples:
  • First create two sets:
    | set1 set2 |
    set1 := Set new add: 3; add: 4.
    set2 := Set new add: 4; add: 5.

    Now with variables set1 and set2 you can evaluate:
  • set1 intersection: set2 returns Set( 4 ).
  • set1 difference: set2 returns Set( 3 ) (set1 minus set2).
  • set1 union: set2 returns Set( 3 4 5 ) (no duplicates).
Set implements the full JS class Set, that strangely also has map-like behavior.
That functionality is not detailed here.
