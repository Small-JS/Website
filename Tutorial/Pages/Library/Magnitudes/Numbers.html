Numbers
SmallJS implements the 'traditional' Smalltalk number hierarchy,
that is different from JS, but is more advanced and consistent.
Number classes are: Integer, BigInt, Float, Fraction.

When performing calculations with different number types,
the return type will be the 'bigger' type that can accomodate the result, see below.

Numbers can be converted to other number types
with the method named 'to' followed by a number class name.
E.g.: 3.14 toInteger returns 3.
Integer
The class Integer implements fixed-size integers,
that are in the range of JS 'safe' integers: -2^53+1 to +2^53-1.

Most calculations are implemented with familiar operator methods like: +, -, *, /.
Integer division and remainder are implemented with // and % respectively.

Examples:
  • 3 + 4 returns 7.
  • 6 / 3 returns 2.
  • 7 // 3 returns 2.
  • 8 % 3 returns 2.
BigInt
The class BigInt implements integers with arbitrary sizes.
When the result of an Integer calculation is outside its range,
the result is automatically converted to a BigInt.
BigInt implements all common calculation methods of Integer.

Examples:
  • 20 factorial returns a BigInt with value 2432902008176640000.
  • 999999999 squared returns a BigInt with value 999999998000000001.
Fraction
The class Fraction implements fractional numbers with exact precision,
by storing the numerator and denominator separately.
When the result of a an integer division / is not a whole number,
the result is automatically converted to a Fraction.
Fractions are automatically normalized to the smallest numerator and denominator
with the same fractional value.

Examples:
  • 7 / 3 returns a Fraction with value ( 7 / 3 ) (seven thirds).
  • 7 / 3 * 3' returns exactly 7 (no rounding errors).
  • ( 1 / 4 ) + ( 1 / 4 ) returns ( 1 / 2 ).
  • ( 2 / 3 ) numerator returns 2.
  • ( 2 / 3 ) denominator returns 3.
Float
The class Float implements floating point numbers, in the range +/- ~ 1.79 * 10^308.
A Float instance is created using a decimal point, e.g.: 3.14
Mixed calculation of a Float with another number type will return a Float.

In addition to the common math operations mentioned above,
Float implements these well known rational number operations:
sqrt, ** (power), exp, ln, log, sin, cos, tan.
The Float class has these methods that create new numbers:
pi (3.14...), random (between 0 and 1)

Examples:
  • 1.2 + 3 returns 4.2.
  • 2.0 sqrt returns 1.4142....
  • Float pi returns 3.14....
