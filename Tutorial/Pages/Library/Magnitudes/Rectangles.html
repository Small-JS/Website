Rectangles
The class Rect implements rectangles with an origin and an extent .
The origin is a Point specifying its top-left corner.
The extent is a Point specifying its width (x) and height (y).
Rectangles can be compared for value equality.

When implementing HTML DOM functionality, SmallJS always passes Rect objects
in stead of passing separate x, y, width and height coordinates.
Rectangles are mostly used by the Canvas HTML element.

Examples:
  • Rect origin: 8 @ 3 extent: 4 @ 2 returns a rect Rect( ( 8 @ 3 ) , ( 4 @ 2 ) ) .
  • ( Rect origin: 8 @ 3 extent: 4 @ 2 ) origin returns a point ( 8 @ 3 ) .
  • ( Rect origin: 8 @ 3 extent: 4 @ 2 ) extent returns a point ( 4 @ 2 ) .
  • ( Rect origin: 8 @ 3 extent: 4 @ 2 ) = ( Rect origin: 8 @ 3 extent: 4 @ 2 )
    returns true .
