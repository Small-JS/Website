Strings
Strings are implemented by the class: String .
String constants are written between single quotes: '' .
(Double quotes identify code comments: "This method does X" )
Special characters in a string constant can be 'escaped' with a backslash: \ .

SmallJS implements all JS string operations.
String concatination can also be done with a comma: , (next to using + ) .

Examples:
  • 'abc' returns String instance 'abc'
  • 'ab', 'de' returns 'abcd'
  • 'a\tb' returns 'a    b'
  • 'abc' length returns 3
  • 'abcd' substring: 1 to: 3 returns 'bc'
