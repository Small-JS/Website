Points
The class Point implements 2D points with x and y coordinates.
Points can be constructed conveniently by calling the @ method on a number.
When comparing points, the comparison is applied to its coordinates with boolean 'and'.
Calculations on points are implemented with familiar operator methods like: +, -, *, /.
These calculations can be done with two points or with a point and a number.

When implementing HTML DOM functionality, SmallJS always passes Point objects
in stead of passing separate x and y coordinates. E.g. see class Rect .

Examples:
  • 2 @ 3 returns a point ( 2 @ 3 ) .
  • Point x: 2 y: 3 also returns a point ( 2 @ 3 ) .
  • ( 2 @ 3 ) x returns 2 .
  • ( 2 @ 3 ) y returns 3 .
  • ( 2 @ 3 ) = ( 2 @ 3 ) returns true .
  • ( 2 @ 3 ) < ( 4 @ 3 ) returns false (y is not smaller).
  • ( 2 @ 3 ) + ( 4 @ 5 ) returns ( 6 @ 8 ) .
  • ( 2 @ 3 ) * 2 returns ( 4 @ 6 ) .
Point3d
The class Point3d implements 3D points, adding a z coordinate.
Class Point3d inherits from class Point implementing all its functionality.
Point3d objects can be constructed conveniently by calling the @ method on a 2D point.
On operations with a Point3d and a 2D Point,
the 2D point is assumed to have a z coordinate of 0.

Examples:
  • 2 @ 3 @ 4 returns a 3D point ( 2 @ 3 @ 4 ) .
  • ( 2 @ 3 @ 4 ) z returns 4 .
  • ( 2 @ 3 @ 0 ) = ( 2 @ 3 ) returns true (z defaults to 0).
