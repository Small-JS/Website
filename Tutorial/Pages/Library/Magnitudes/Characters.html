Characters
Characters are implemented by the class: Character .
Character constants are written with a preceeding dollar sign $ .
When calling the index method at: on a string, a character is returned.
(So not a string of length 1, like in JS)
You can create a character from a unicode value with the class method: fromCode: .
These methods test for types of characters:
isLetter, isUppercase, isLowercase, isDigit .

Examples:
  • $a returns character a .
  • 'abc' at: 1 returns b
  • Character fromCode: 64 returns @ .
  • $7 isDigit returns true .
