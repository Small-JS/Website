Objects
In Smalltalk, every value is an object.
And every object is an 'instance' of a class (template).

Examples for immediate object instances:
- 42 is an instance of class Integer.
- 'Hello' is an instance of class String.
- true is an instance of class Boolean.
- $a is an instance of class Character.
- 3.14 is an instance of class Float.
- nil is the singleton instance of class Nil.
- self is the current object instance.

You can check this by evaluating 42 class in the Playground.
The result should be class Integer.
