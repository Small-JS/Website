Blocks

Smalltalk blocks contain statements that are not executed right away.
Blocks are equivalent to anonymous (lambda) functions in other languages.
They are also used to implement all control flow statements like if-then, do-while.
Blocks are delimited with square brackets: [] .
A block returns the value of the last expression inside it.

Examples:
 - 1 + 2 < 4 ifTrue: [ 'smaller' ] returns 'smaller'
- 1 + 5 < 4 ifTrue: [ 'smaller' ] ifFalse: [ 'bigger' ] returns 'bigger'

Blocks with arguments

Blocks can have one or more arguments (parameters)
A parameter variable name is preceded by a colon :.
A vertical bar | separates the arguments from the block statements.

Examples:
 - 1 to: 4 do: [ :i | Console log: i ]
  Logs the numbers 1 though 4 on the console.
- #( 5 6 7 ) find: [ :num | num even ]
Returns 6 (first even number)
- Dictionary new do: [ :key :value |
    Console log: key, ': ', value ]
  Logs the keys and values of an (empty) dictionary.
