Statements & Variables

Statements

Smalltalk separates expressions into statements with a period.
(In most languages the semicolon is used for this)

Example:
 1 + 2.
3 + 4.

This will return 7, the value of the second statement.
Note: Smalltalk is not sensitive to newlines.
Putting these statements on a single line will give the same result.

Variables

Smalltalk has three types of variables.
Local variables

Local variables are declared at the start of a method or block between two vertical bars.
The default value of a variable is the nil object.
Variables can be assigned to other object with :=.

Example:
 | num | num := 7. num := num * 2.
This will set variable num to 14.

Argument variables

Agrument variables are declared with method declarations.
They can then be used only within that method (local scope).

Example method:
 isLegalAge: person
  ^ person age >= 18.
This method will return true if the argument person is at least 18 years old.
Note: Adding methods is not supported in the Playground.
We'll come back to this when editing classes later.

Global variables

In SmallJS, the only global variables are singleton class instances, like we saw before, e.g.: Integer, String or Boolean.

If you want to store a custom global variable, your can add it as a 'class variable' to a class.
More on that later.
