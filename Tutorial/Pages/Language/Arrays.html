Arrays

Basic operations
Arrays contain collections of element objects.
Arrays can be defined in code with #( ... ),
using space separated objects or expressions within the parentheses.
Arrays can be grown in size, using add: to add an element.

Array elements can be:
- Retrieved with a zero-based index using at:.
- Replaced using at:put.
- Iterated over using do:.
- Converted into a new array using map:.

Examples:
 - #( 3 'd' ( 3 + 0.5 ) ) returns the array object #( 3 d 3.5 ).
- #( 3 4 ) add: 5 returns #( 3 4 5 ).
- #( 3 4 5 ) at: 1 returns 4.
- #( 3 4 5 ) at: 1 put: 6 returns #( 3 6 5 ).
- #( 3 4 5 ) do: [ :e | Console log: e ] logs elements 3, 4 and 5.
- #( 3 4 5 ) map: [ :e | e squared ] returns #( 9 15 25 ).

Typed arrays

SmallJS also implements some JS typed arrays.
There arrays store one specific data type iso arbitrary ST objects.
For handling binary data there are Uint8Array and ArrayBuffer.

Other collection classes

SmallJS also implements these other collection classes:
- Dictionary : A key-value dictionary for arbitrary objects.
- Map : An efficient key-value dictionary with string keys.
- Set : Stores unique object values and supports mathematical set operations.
