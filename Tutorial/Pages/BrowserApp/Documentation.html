Documentation
As a last, optional step, lets create some documentation for our app
We'll use the Markdown standard for it.

First rename the file ./Counter.md to ./Frequencies.md .
Then replace its contents with this markdown text:
# Frequencies tutorial app

This project is a SmallJS implementation of a simple letter freqency counter app,
that displays the results in a HTML table.

## Running

First run `./startWebServer.sh`.
Then start the project in VSCode with `F5`.


Congratulations, you've finished the 'Browser App' tutorial.
Now you can make your own cool browser apps in Smalltalk! :-)
